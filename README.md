# Viral Dedaniya — Portfolio (Next.js + Tailwind)

## Quickstart
```bash
npm i
npm run dev
# open http://localhost:3000
```

## Deploy (Vercel)
- Push this repo to GitHub.
- Import into Vercel and deploy.
- Your resume is bundled at `/public/resume.pdf` → site serves `https://<domain>/resume.pdf`.

## Edit Content
- All text and sections live in `app/page.tsx` inside the `DATA` object.
- Update project `code` and `demo` URLs when you have them.

## Stack
- Next.js 14 (App Router), React 18
- Tailwind CSS (class-based dark mode)
